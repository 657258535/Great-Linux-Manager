# Great-Linux-Manager

Linux文件管理系统，傻瓜式操作，一秒上手，简洁的外表希望能得您的青睐


### 使用方法：

> 1.上传至网站根目录

> 2.关闭防跨站，如果只管理本站数据，忽略此条

> 3.index.php第30行配置登录账号密码

> 4.尽情使用
